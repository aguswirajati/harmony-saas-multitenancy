# Contributing to Harmony## Getting Started1. Fork the repository
2. Create feature branch: `git checkout -b feature/my-feature`3. Commit changes: `git commit -m 'feat: add some feature'`4. Push to branch: `git push origin feature/my-feature`5. Open Pull Request
## Code Style- Backend: Follow PEP 8 (Python)
- Frontend: Follow Airbnb style guide (TypeScript)
- Use conventional commits
## Testing- Write tests for new features
- Run tests before submitting PR: `pytest`## Questions?Open an issue or discussion
